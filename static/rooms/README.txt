To add room images:
1. Create folders for each room type (1BBFG, 1BGS, etc.)
2. Add numbered images (1.jpg, 2.jpg, 3.jpg, etc.) in each folder
3. Update the count in src/lib/utils/room-images.ts if needed

Example structure:
static/rooms/1BBFG/1.jpg
static/rooms/1BBFG/2.jpg
static/rooms/1BGS/1.jpg
...
